# mruna-designs

mruna-designs is a modern ecommerce frontend for dresses, t-shirts, and pants built with Angular.

## Features
- Shop by category with curated product collections
- Product detail pages with variant selection and cart actions
- Cart, checkout, account, and admin-friendly routes
- Responsive layout with accessible UI patterns

## Development

```bash
npm install
npm start
```

Once the server is running, open your browser and navigate to http://localhost:4200/.
