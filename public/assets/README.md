# Assets Directory

This directory contains static assets for the application. The following files are referenced in the app:

## Required Files

- **background.jpg** - Background image for the page (referenced in app.css)
- **tree.png** - Tree image displayed on the page (referenced in app.html)
- **music.mp3** - Audio file played when the button is clicked (referenced in app.ts)

## Instructions

Replace the placeholder files in this directory with your actual assets:
1. Add your background.jpg file
2. Add your tree.png file  
3. Add your music.mp3 file

The files are served as static assets from the public directory and will be accessible at:
- /assets/background.jpg
- /assets/tree.png
- /assets/music.mp3
